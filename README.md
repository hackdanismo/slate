# Slate

## What is Slate?
`Slate` is a collection of reusable UI components for web pages and applications built using: `HTML`, `CSS` and `JavaScript` and is released as an open-source project under the [MIT licence](https://github.com/hackdanismo/slate/blob/main/LICENSE). Think of it as a similar project to [ShadCN](https://ui.shadcn.com/) but using vanilla components.
